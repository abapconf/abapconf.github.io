Agenda: SAP Inside Experience

Agenda for room W3. This room is located at the 1st floor.

Sessions start at 10:15! The welcome and keynote is in the Audimax (ground floor). The sessions in room W3 end at 18:00.

SAP Show and Tell with experts from SAP that will present best practices from their area of expertise. The topics are practically presented with a mixture of lecture, demo and code examples. There is space for attendees to ask questions and exchange ideas with the SAP experts.

10:15 - 11:00

Core Data Services - Deep dive on new features and capabilities

W345 minutes Language: EN
Download: PDF
In the last 10 years, ABAP CDS has revolutionized the data modeling layer for ABAP applications. New features are constantly being added to establish ABAP CDS as the gold standard for data modeling in the ABAP Cloud development model. This session will provide an overview of the current feature scope of ABAP CDS with an emphasis on recent developments. Learn about user-defined CDS types, CDS scalar functions, CDS entity buffers, and much more! Including live demo.
Andrea Schlotthauer
Konrad Gaerdes

Advanced ABAP - A practical Showcase

W345 minutes Language: EN
In this session, we'll explore recent developments in ABAP programming, focusing on features such as constructors and advanced table operations. Through practical examples, we'll demonstrate how these enhancements can improve code efficiency and streamline development tasks. We'll also discuss potential challenges and share best practices to help attendees successfully implement these features in their projects.

Keynote: ABAP Cloud in action – Latest features and capabilities

Audimax45 minutes Language: EN
The ABAP Cloud development model supports the ABAP developer to build applications, extensions and services for the cloud. New ABAP Cloud apps can be automatically generated using Generative AI capabilities embedded into the ABAP Development Tools for Eclipse. The generated apps have built-in ABAP Cloud qualities such as end-to-end extensibility for side-by-side and on-stack extension scenarios.

The session will show how to integrate business events and expose the underlying data model for Enterprise Search and Analytics use cases.
Volker Drees

Embedded Analytics

Download: PDF

ABAP custom code transformation for Clean Core in SAP S/4HANA

W345 minutes Language: EN
In this session you will learn how to handle your ABAP custom code after an SAP S/4HANA transition with regard to clean core including different options: from retirement of the unused code and adaptation for SAP S/4HANA to renovation and innovation with ABAP Cloud. The ABAP test cockpit on SAP BTP will support the custom code transformation and management in your SAP S/4HANA system landscapes.
Olga Dolinskaja
