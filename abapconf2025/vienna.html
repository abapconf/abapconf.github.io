Location: Vienna

Welcome to the capital auf Austria: Vienna! A place to visit for everyone interested in history, culture, architecture. V5 - CDS-only Data Modeling - Harnessing New ABAP CDS Object Types for Modern RAP Applications

Vienna 60 minutes Language: EN
With the continuous evolution of ABAP, the introduction of new Core Data Services (CDS) object types in recent releases has revolutionized the way developers model and interact with data. This session will provide a comprehensive overview of the latest additions to the ABAP CDS ecosystem, including CDS Table Entities, CDS Aspects, CDS Scalar Functions, CDS Types, CDS Entity Buffers, and CDS External Entities, with a focus on their role in the RESTful Application Programming (RAP) model.

Key Topics:

1. CDS Table Entities
2. CDS Aspects
3. CDS Scalar Functions
4. CDS Types
5. CDS Entity Buffers
6. CDS External Entities

Bringing It All Together: The core focus of this session is how all these CDS object types can be combined within the context of RAP (RESTful ABAP Programming Model) to build efficient, scalable, and cloud-ready applications in the ABAP ecosystem.
Katharina Altrichtery
Matthias Herchenröther

V5: Smart ABAP: Design Patterns for Real-World Scenarios

Vienna 60 minutes Language: EN
This session explores essential design patterns in ABAP, focusing on practical applications for writing clean, scalable, and maintainable code. For example, by implementing the MVC (Model-View-Controller) pattern, we can seamlessly switch from cl_salv_table to cl_gui_alv_grid when requirements change to support editable ALV, without major code rewrites. I want to present 3 or even more real-world scenarios and then the thought process of choosing the right design pattern for each case.
George Drakos

V13: The Magical World of SAP EWM for ABAPers

Vienna 60 minutes Language: EN
Explore the fascinating and confusing world of SAP Extended Warehouse Management (EWM). Its back end is an eclectic blend of old and new technologies, unlike anything ABAP developers have encountered before. In this session, you'll learn about EWM’s peculiarities, main processes, enhancement options, and more. Don’t fall into the trap of thinking, “It’s just ABAP - how hard can it be?” Be prepared!
Jelena Perfiljeva

V14: Effortless ABAP dependency management with TRM

Vienna 60 minutes Language: EN
This talk dives into how TRM (shown in a previous demo) simplifies ABAP dependency management by automating the resolution and installation of required libraries. We will demonstrate how to declare and implement dependencies during development and showcase the deployment process on a target system, where TRM ensures seamless package installation.
